# Learning Git and GitHub

A comprehensive guide to becoming a power user: master Git, GitHub, command line, VS Code, Docker, SSH, networking, and more. Everything you need to work efficiently as a developer in one place.

## Quick Start

- **New to GitHub?** Start with [`docs/getting-started.md`](docs/getting-started.md) — step-by-step guide to creating and pushing your first repository.
- **Learn Markdown on GitHub** Read [`docs/markdown.md`](docs/markdown.md) — comprehensive guide to GFM syntax with practical examples.
- **New to the command line?** Check [`docs/cli-basics.md`](docs/cli-basics.md) — terminal essentials for any OS.
- **Want to contribute?** See [`CONTRIBUTING.md`](CONTRIBUTING.md) for guidelines and workflow.

## Topics Covered

### Version Control & Collaboration

- Writing clear, engaging README files
- GitHub Flavored Markdown (GFM) syntax and best practices
- Git branching, merging, and version control
- GitHub collaboration features (issues, pull requests, discussions)
- Creating and using templates
- Publishing docs with GitHub Pages
- Setting up CI/CD for documentation

### Development Environment

- Visual Studio Code setup and customization
- Essential VS Code extensions
- Docker containerization basics
- Command line proficiency (PowerShell and Bash)
- Keyboard shortcuts for productivity

### Infrastructure & Networking

- SSH key management and secure remote access
- File transfer with SCP, SFTP, and Rsync
- Networking fundamentals (IP, DNS, ports, protocols)
- Server basics and common services
- Cloud computing concepts

## Getting Started

1. Clone or fork this repository
2. Read [`docs/getting-started.md`](docs/getting-started.md) to learn the basics
3. Explore [`docs/cli-basics.md`](docs/cli-basics.md) for command line fundamentals
4. Set up [`docs/vscode.md`](docs/vscode.md) for an efficient development environment
5. Check out the `examples/` folder for real-world README samples
